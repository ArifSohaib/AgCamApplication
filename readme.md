How to install: <bn>

Type the following in your Raspberry Pi Termainal


clone the repo
```
git clone https://github.com/ArifSohaib/AgCamApplication.git
```

Install the required libraries
```  
python3 -m pip env install 
```

allow it to run as an executable
```
chmod +x main.py
```

Start the app
```
.\main.py
```
